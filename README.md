# LocalHub UI 업데이트

## 반영 사항
1. 게시판 목록, 게시글 상세, 글 작성 화면의 뒤로가기 버튼을 각 화면 좌상단에 통일했습니다.
2. 장소 팝업과 비밀번호 팝업에도 좌상단 뒤로가기 버튼을 적용했습니다.
3. 전체 색상 체계를 CSS 변수 기반으로 재구성했습니다.
4. 상단 오른쪽에 라이트/다크 모드 전환 버튼을 추가했습니다.
5. 선택한 테마는 `localStorage`의 `localhub-theme`에 저장됩니다.
6. 서버 요청, 지도 초기화, 댓글, 게시글, 챗봇 오류는 우측 상단 토스트 팝업으로 표시됩니다.
7. 토스트는 3초 뒤 자동으로 사라지며, 사용자가 `×` 버튼으로 즉시 닫을 수도 있습니다.
8. 게시글 및 댓글 등록, 수정, 삭제 성공 시에도 성공 토스트가 표시됩니다.

## 파일
- `index.html`
- `app.js`
- `style.css`
- `config.js`

## 실행
`config.js`의 `API_BASE_URL`을 서버 주소에 맞춘 뒤 정적 서버로 실행하세요.
